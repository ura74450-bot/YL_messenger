# 💬 YL Messenger v14

Мессенджер с поддержкой групп, каналов, голосовых сообщений, медиафайлов и статусов.

## Возможности

### Переписки
- 💬 Личные сообщения (DM) между пользователями
- 👥 Групповые чаты с участниками
- 📢 Каналы с подсчётом просмотров
- 🔍 Поиск пользователей и каналов/групп на сервере
- ✅✅ Синие галочки — сообщение прочитано
- ⌨️ Индикатор печати в реальном времени

### Медиа
- 🖼 Отправка фото и видео
- 📎 Прикрепление файлов
- 🎤 Голосовые сообщения (удерживай кнопку)
- 📤 Прогресс загрузки виден у получателя
- ▶️ Воспроизведение голосовых с таймером обратного отсчёта

### Аккаунты
- 👤 Несколько аккаунтов на одном устройстве (полная изоляция переписок)
- 🖼 Обновление аватарки без лага статуса онлайн
- 🔔 Уведомления с аватаром и именем отправителя
- 🚫 Блокировка пользователей
- 📸 Статусы (видео до 30 сек, исчезают через 24ч)

### Безопасность
- Имя отправителя берётся с сервера — нельзя написать от чужого имени
- История чатов изолирована между аккаунтами

## Запуск локально

```bash
node server.js
# Открой http://localhost:3001
```

## Деплой на Render

1. Загрузи папку на GitHub
2. [render.com](https://render.com) → **New Web Service** → выбери репо
3. **Start command:** `node server.js`
4. После деплоя вставь URL в `index.html`:
   ```js
   const SERVER_URL = 'https://your-app.onrender.com';
   ```

## Стек

- **Frontend:** Vanilla JS, HTML/CSS (один файл `index.html`)
- **Backend:** Node.js без зависимостей (чистый HTTP + WebSocket RFC 6455)
- **Хранение:** `data.json` (персистентное) + `localStorage` (на клиенте)
